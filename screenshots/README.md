# Screenshots Directory

## Required Screenshot Files

Save your application screenshots in this directory with these exact names:

1. **main-view.png** - Full application view showing event list and ECG plot area
2. **event-list.png** - Close-up of event list sidebar showing multiple events
3. **afib-event.png** - AFIB event ECG plot with red event marker
4. **vtach-event.png** - VTACH event ECG plot with red event marker
5. **brady-event.png** - BRADY event ECG plot with red event marker
6. **event-info.png** - Event information panel with metrics
7. **multiple-events.png** - Sidebar showing different event types
8. **api-docs.png** - API documentation (Swagger UI) page

## How to Add Images

1. Take screenshots of your application
2. Save them to this `screenshots/` directory
3. Name them exactly as listed above
4. Commit and push:
   ```bash
   git add screenshots/*.png
   git commit -m 'Add application screenshots'
   git push origin main
   ```

The images will automatically display in the README.md file on GitHub.
