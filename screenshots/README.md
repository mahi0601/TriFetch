# Screenshots Directory

Place your application screenshots in this directory.

## Required Screenshots

1. **main-view.png** - Full application view showing event list and ECG plot
2. **event-list.png** - Close-up of the event list sidebar
3. **ecg-plot-afib.png** - ECG plot for an AFIB event with red marker
4. **ecg-plot-vtach.png** - ECG plot for a VTACH event with red marker
5. **ecg-plot-brady.png** - ECG plot for a BRADY event with red marker
6. **event-info.png** - Event information panel
7. **multiple-events.png** - Showing different event types
8. **api-docs.png** - API documentation page

## File Naming

Use the exact filenames listed above so they display correctly in the README.

## Image Format

- PNG or JPG format
- High resolution (at least 1920x1080 recommended)
- Clear and readable text
- Red event marker clearly visible in plot screenshots
