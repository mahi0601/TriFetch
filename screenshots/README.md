# Screenshots Directory

Place your application screenshots in this directory with the following names:

## Required Screenshots

1. **main-view.png** - Full application view showing event list and ECG plot
2. **afib-event.png** - AFIB event ECG plot with red event marker
3. **vtach-event.png** - VTACH event ECG plot with red event marker
4. **brady-event.png** - BRADY event ECG plot with red event marker
5. **event-info.png** - Event information panel
6. **multiple-events.png** - Multiple event types in sidebar
7. **api-docs.png** - API documentation page

## How to Add Images

1. Save your screenshot images to this directory
2. Name them exactly as listed above
3. Run: `git add screenshots/*.png && git commit -m 'Add screenshots' && git push origin main`

The images will automatically display in the README.md file on GitHub.
